# voidfetch

minimal system info.

## install

### linux / macos

```sh
git clone https://github.com/AstralZX/voidfetch.git
cd voidfetch
chmod +x install.sh
./install.sh
```

### windows (powershell)

```powershell
git clone https://github.com/AstralZX/voidfetch.git
cd voidfetch
.\install.ps1
```

## usage

```sh
voidfetch
```
